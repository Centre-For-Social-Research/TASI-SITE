# Design Rules

- If a box, card, panel, rectangle, input, or any similar boxed UI element is used, the corner radius must be `10px` only.
- Do not use border radius values greater than `10px` for box-style design elements.
- This rule applies across the site to all boxed design components unless explicitly overridden by the project owner.
- Keep buttons as they are unless a button-specific change is explicitly requested by the project owner.
